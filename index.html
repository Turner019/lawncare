Setup Required
Configure your credentials before deploying
1. supabase.com → New Project → Settings → API
Copy Project URL and anon public key → paste into SUPABASE_URL and SUPABASE_ANON at the top of this file's script.
2. Run schema.sql (included) in Supabase → SQL Editor.
3. app.regrid.com → free trial → copy your API token → paste into REGRID_TOKEN . Free tier: 2,000 parcel lookups / 30 days.
4. Push index.html , manifest.json , sw.js to GitHub Pages.
